# People **Powered** Politics

> Democracy doesn't end on election day. P3 makes it continuous.

**P3** is a privacy-first civic participation platform that gives citizens a secure, anonymous way to have a real say in policy — not once every four years, but every day. Built on Cardano and Midnight blockchain infrastructure.

---

## What P3 does

**Standard layer — Government → Citizen**
Active policy proposals appear in the P3 app. Citizens vote anonymously. Policy makers receive anonymised participation data via a secure dashboard. No names, no profiles, no personal data stored — ever.

**Parliament 3 — Citizen → Government**
Citizens open topics, debate, and refine societal issues through collective intelligence. When a proposal is ready, top contributors vote. At 70% approval, it travels via smart contract directly to the relevant government body. What politicians can do, citizens can do too.

## Tech stack

| Layer | Technology |
|---|---|
| Identity & privacy | Hyperledger Identus (SSI · DID · ZKP) |
| Private compute | Midnight sidechain · Compact smart contracts |
| Public audit anchor | Cardano mainnet · Aiken validator |
| Mobile app | React Native |
| API | REST + GraphQL |
